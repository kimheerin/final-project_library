희망도서 신청은 어떻게 하나요?

1. 희망도서 신청방법 : 홈페이지 로그인- 자료검색 - 희망도서신청 - 신청하기
2. 희망도서 신청 권수 : 1인당 월 3권
3. 희망도서 접수 : 매주 월(전주 월~일 신청도서) 

선정제외도서(자세한 내용은 홈페이지 희망도서 신청 안내 참조) 
- 비도서, 서양서, 만화류
- 만화, 무협지, 라이트 소설, 판타지 및 인터넷 소설
- 시험 및 자격증 취득을 위한 수험서
- 교재, 문제집, 전집류
- 5년 이전 발간도서
- 고가의 도서(5만원 이상)
- 기관의 자료선정기준에 맞지 않는 도서 
<희망도서 처리> 
- 희망도서는 매주 월 처리하고 1인당 월 3권까지 신청가능
- 희망도서 선정 결과 홈페이지 신착자료목록에 게시
- 희망도서 취소 사유 희망도서 신청 시 입력한 메일로 안내
- 희망도서 신청 시 입력된 핸드폰으로 대출 가능함을 알려주는 문자메시지 전송
- 희망도서 비치기간 : 연락 후 3일 동안 별도 비치(자료실 직원에게 문의)

무선인터넷(Wi-Fi)는 어떻게 이용하나요?

서울특별시교육청 정보보안 기본지침 제46조(무선랜 보안관리)강화 및 통합도서관 ID/PW 입력 간소화에 따라

2018.4.2.(월)부터 도서관 무선 네트워크(Wi-Fi)연결 및 인증 방식이 변경되었습니다.

변경 된 연결 및 인증 방식을 안내 해 드리니 이용에 참고하시기 바랍니다.

로그인 오류가 발생한 경우

홈페이지 보안강화로 2012.11.20(화)부터 windows XP 이용자는 서비스팩3을 설치해야 로그인이 가능하며 오류발생 시 다음을 참고하여 업그레이드 하시기 바랍니다.
 
◆ 오류발생 PC : Windows XP SP2(서비스팩2) 이하 버전의 PC
◆ 오류현상 : "웹페이지를 투시할 수 없습니다" 라는 메시지 보임
◆ 조치방법 : Windows XP SP3(서비스팩3)로 업그레이드
◆ Windows XP SP3 설치방법
 → Windows Update 및 다운로드 센터에서 수동 설치
 [windows Update 사이트]
  http://update.microsoft.com/
 [서비스팩 다운로드센터]
  http://windows.microsoft.com/ko-KR/windows/downloads/service-packs
◆ 서비스팩 확인 방법 : 내컴퓨터 → 속성 → 확인

회원 정보 수정 방법

홈페이지 내 회원 정보 수정은 나의책숲-내정보수정 메뉴에서 가능합니다.

홈페이지에서 이용자가 직접 수정 가능한 개인정보는 이름, 비밀번호, 이메일주소입니다.

도서 대출 방법

도서 대여는 홈페이지 로그인 후 상단 메뉴인 자료검색에서

원하는 책을 검색 후, '대여' 버튼을 클릭합니다.

그동안 대출했던 도서 목록 확인

이전 대출 내역 확인은 홈페이지 상단 메뉴인 나의책숲-내서재 메뉴에서 확인 가능합니다.

찾고 싶은 도서 검색

도서 검색은 홈페이지 상단 도서검색-자료검색 메뉴에서 자료를 확인해보실 수 있습니다.
